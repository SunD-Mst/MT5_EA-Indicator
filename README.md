## 🧠 策略内核与波动率哲学

本仓库的策略虽然形态各异（突破、回归、网格），但它们共享同一套**波动率驱动的决策框架**：

- **趋势策略**：依赖波动率扩张确认突破的有效性，试图捕捉动量爆发。
- **回归策略**：依赖波动率极值识别价格偏离，博弈均值回复。
- **网格策略**：依赖波动率百分位动态调整间距，在低波区蓄力，高波区避险。

这些逻辑已在实盘中证明了它们的生命力，但请记住：**没有任何一个参数设置是永恒有效的。** 它们需要你根据当下的市场微观结构去校准。

## ⚙️ 安装指南

1. 将 `Experts` 文件夹内的 `.mq5` 文件复制到 MT5 数据目录 `MQL5/Experts/`。
2. 将 `Indicators` 文件夹内容复制到 `MQL5/Indicators/`。
3. 若有 `Include` 文件夹，复制至 `MQL5/Include/`。
4. 重启 MT5 终端，在 MetaEditor (`F4`) 中编译所有文件。
